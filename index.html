  • üìÇ AWS S3 Bucket Terraform

    This project showcases how to automate the setup of a static website using Terraform and Amazon S3. The Terraform configuration handles the creation of an S3 bucket, sets it up for static website hosting, and configures public access so users can view the website directly via the S3 endpoint. With a few simple Terraform commands, the infrastructure is deployed automatically, allowing you to focus on your content instead of manual setup. After deployment, you can upload your HTML, CSS, and JavaScript files to the S3 bucket, and your static website is live.

    Key Features:

    • Infrastructure as Code with Terraform
    • Static website hosting on AWS S3
    • Public access configuration
    • Easy deployment and cleanup

    How to Use:
    • Clone the repo:
      • git clone https://github.com/ShubhamMca88/aws-s3-bucket-terraform-project.git
      cd aws-s3-bucket-terraform-project
    • Initialize Terraform:
      • terraform init
    • Review changes:
      • terraform plan
    • Apply configuration:
      • terraform apply
    • Upload your website files to the S3 bucket manually after creation.
    • Clean up when done:
      • terraform destroy
